About Me

My name is Jan Bertlik. I was born and raised in Toronto Canada, to a Czech - Slovak family. I have been fortunate enough to grow up in a time when silicon transistors were rapidly evolving computers, and with them; the web.

Since the 90's, I've been surfing the internet which went hand in hand with my interest in technology. Some time ago, I decided that I would like computers to be in my future.

Wanting to take a less common approach, I decided to take my interest in technology more seriously and attended a 12 week boot camp in Toronto focusing on web development. Here we learned about the MVC model with the Ruby on Rails framework for backend work, while adding some Javascript spice to make our frontend interactive.

After this bootcamp, I was interested in learning another OOP language that was more versitile. Python was the best choice for me since the syntax is similar to Ruby, and it is widely used in machine learning and data science (something I would like to take the time to learn some day). Following many tutorials, I finally built this site using the Flask framework. Flask was chosen over Django at first because I read that Flask is more 'Pythonic' and easier to learn. You can read more about this site and take a look at the code here.
